[22기 이세훈 DL study 1주차 과제]

1. 선택한 AI 패턴 : “Predictive Analytics & Decisions”

2. 생각해본 적용 방안
- 문제: 최근 SNS나 검색엔진의 결과에 선정적인 / 불법적인 광고나 바이럴마케팅이 많아짐 -> 일반 소비자 효용 저해
- 해결: 각각의 플랫폼에서 인위적인 anomaly를 탐지하고 차단해내면 좋을 것 같다

3. 모델의 입/출력
참고자료: ’SNS 텍스트 기반 마약 의심 거래 탐지 알고리즘에 대한 연구: 패턴 및 맥락분석을 중심으로”
(Doi: 10.22987/jifso.2025.16.1.172)

YouTube 운영자 기준으로 생각: 
선정적인 댓글 / 불법 광고 신고된 댓글과, 그것의 label을 처리한 데이터가 있을 것
>> 이를 사용하여 탐지 대상 댓글에 자주 쓰이는 단어 / 그렇지 않은 단어를 구분
(참고논문에서는 LIME: Local Interpretable Model-agnostic Explanations 사용)
>> word2vec 형식으로 벡터 공간 매핑

최종 모델 (classification)
입력: 새로운 유튜브 댓글들
출력 : 비정상적 댓글이면 1, 정상적 댓글이면 0
